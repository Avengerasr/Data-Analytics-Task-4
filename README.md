# Sales Performance Dashboard

## Objective
Analyze sales performance and customer behavior using Power BI.

## Dashboard Insights
- Dinner generates highest revenue (~76%)
- Weekends drive maximum sales
- Average tip is ~16%
- Group size 2 contributes most revenue

## Data Story
Included in data_story.docx

## Hypothesis Testing

Hypothesis:
Dinner generates higher revenue than lunch.

Test Used:
Two-sample T-test

Results:
t-value = 2.93  
p-value < 0.01  

Conclusion:
Dinner generates significantly higher revenue than lunch.
